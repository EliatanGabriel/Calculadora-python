# 🧮 Calculadora em Python

## 📌 Descrição
Calculadora desenvolvida em Python com interface de linha de comando (CLI), permitindo realizar operações matemáticas básicas com tratamento de erros.

## 🚀 Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Tratamento de divisão por zero
- Validação de entrada do usuário

## 🛠 Tecnologias utilizadas
- Python 3

## ▶️ Como executar o projeto

1. Clone o repositório:
git clone https://github.com/seu-usuario/calculadora-python.git

2. Acesse a pasta:
cd calculadora-python

3. Execute:
python src/main.py
